How does it work?
1. Download .py file.
2. Install required packages
3. Run
4. You will see the ability to choose 2 options: 1 share or some shares to calculate. Do as you wish, but: option "1 share" plots histogram of the distribution, after it you need to restart the        script. Option "some shares" does not give any plots.
5. After chooing the method (input 1 for single share and 2 for list of shares):
   If you choose "1" - specify ticker, press enter and specify number of days of history. Done.
   If you choose "2" - specify tickers one by one. If there's no such a ticker on the stock exchange, you will see an alert and have to restart the script.
6. You see the output of the program in text format

What methods does it have?
1. Historical
2. Delta-normal
3. Monte-Carlo with constant volatility assumption
4. Monte-Carlo with GACH-modelled volatility
